# 更新日志

所有更新日志按时间倒序排列。

## [v2.0.0] - 2026-05-06
## 版本访问地址：https://rertchat-ai.pages.dev/

### ✨ 对话URL分享机制

- **对话ID重构**
  - 将对话ID从随机字符串改为6位数字ID（100000-999999）
  - 新生成的对话自动分配唯一6位数ID
  - 导入的对话也会重新分配新的6位数ID
  
- **URL参数访问**
  - 支持通过 `?chat=123456` 直接访问指定对话
  - 兼容旧参数格式 `?conversationId=xxx` 以保持向后兼容
  - 访问带chat参数的URL会自动切换到对应对话

- **无刷新页面切换**
  - 使用 Next.js 客户端路由实现流畅的对话切换
  - 点击对话列表、新建对话均无需刷新页面
  - URL变化时自动更新当前对话内容

- **注意事项**
  - ⚠️ **分享机制暂未完整实现**：由于对话数据存储在本地 IndexedDB，其他设备访问分享链接时无法获取对话内容
  - 当前仅支持同一设备内的URL跳转
  - 完整的云端分享功能将在后续版本中实现

### ⚙️ 导航设置增强

- **传统导航模式开关**
  - 新增设置项："使用传统对话切换方式"
  - 开启后切换对话不更新URL
  - 关闭后自动使用URL参数方式导航
  - 修复图标重复问题

- **自动跳转优化**
  - 关闭传统导航时，自动跳转到最近对话也会更新URL
  - 确保URL始终与当前对话保持同步

### 💾 设置页面用户体验优化

- **Discord风格保存提示**
  - 修改设置后底部出现醒目的保存提示条
  - 黄色感叹号图标，圆角卡片设计
  - 包含"取消"和"保存"按钮
  - 保存成功后提示条自动消失

- **保存行为优化**
  - 保存设置后不再自动返回主页
  - 保持在设置页面，方便继续配置或核对设置
  - 保存后提示条自动消失，避免用户操作干扰

### 🎨 PWA 主题色优化

- **主题色更新**
  - 将 PWA 主题色从橙色 (#F97316) 改为深色背景色 (#1e2022)
  - 与主页面标题栏背景颜色保持一致
  - 实现沉浸式状态栏效果，使 PWA 应用顶部与系统状态栏融为一体

### 🌙 系统主题支持

- **新增系统主题选项**
  - 在主题设置中添加"系统主题"选项
  - 选择后自动跟随系统设置切换深色/浅色模式
  - 实现真正的自适应主题体验

- **技术实现**
  - 使用 `window.matchMedia('(prefers-color-scheme: dark)')` 检测系统主题
  - 主题类型扩展为 `'light' | 'dark' | 'system'`
  - 支持所有 20 种语言翻译
